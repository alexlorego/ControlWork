Вспомогательные методы:
 InputStringArray - метод для создания массива строк на основе пользовательского ввода с клавиатуры.
В качестве параметра принимает целое число, которое определяет размер создаваемого массива. На выходе
получаем массив из строк.

 InputNumber - рекурсивный метод для проверки пользовательского ввода. Если введенная строка не является целым числом,
выдаст ошибку и запросит повторный ввод.

 Print - метод типа void для объединения массива строк в строку и вывода конечного результата на консоль.

Основной метод:
 GetElementsLowerThen - в качестве параметров принимает массив строк и целое число, определяющее максимальное
количество символов в искомых строках. На выходе из имеющегося массива строк формирует массив строк, длинна которых
меньше либо равна заданного количества символов.
Алгоритм:
1. Начало программы
2. Принимаем исходный массив строк input и целое число maxchar, которое определяет максимальное
количество символов в искомых строках, maxchar = 3
3. Ищем количество элементов массива строк, соответствующих заданным требованиям.
Объявляем переменную count и присваиваем ей значение 0.
Цикл по всем элементам массива input. Если длинна элемента массива меньше либо равно maxchar,
увеличиваем значение count на 1. В конце цилка значение count будет равно количеству элементов массива,
длинна которых меньше либо равна 3.
4. Объявляем новый массив строк result размера count и целое число index, которому присваиваем значение 0. 
Цикл по всем элементам массива input. Если длинна элемента массива меньше либо равно maxchar, присваиваем
элементу массива result с индексом index значение элемента массива с текущим индексом и увеличиваем значение
index на 1.
5. Возвращаем получившийся массив строк result.
6. Конец программы.

Общее решение:
1. При помощи метода InputStringArray формируем массив строк на основе пользовательского ввода данных с клавиатуры.
2. При помощи метода GetElementsLowerThen из имеющегося массива строк формирует новый массив строк, длинна которых
меньше либо равна 3.
3. Выводим получившийся массив на консоль.
